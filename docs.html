1. Fundamentos da Matemática

Números naturais Inteiros Racionais Reais

Conjuntos numéricos

  • Naturais (N): 0, 1, 2, 3...
  • Inteiros (Z): ... -3, -2, -1, 0, 1, 2 ...
  • Racionais (Q): frações e decimais exatos/periódicos.
  • Irracionais: √2, π...
  • Reais (R): união dos racionais e irracionais.

Objetivos em aula

  • Ler, escrever e comparar números.
  • Identificar padrões e regularidades.
  • Resolver problemas do cotidiano.
  • Relacionar cálculo com interpretação.

2. Operações básicas

Adição

Juntar quantidades.

a + b = c

Subtração

Retirar ou comparar.

a - b = c

Multiplicação

Adição de parcelas iguais.

a × b = c

Divisão

Repartição ou medida.

a ÷ b = c

Ordem das operações

Parênteses, potências/raízes, multiplicação/divisão, adição/subtração.

Exemplo

8 + 2 × 3 = 8 + 6 = 14

3. Frações

Conceitos

  • Numerador: partes consideradas.
  • Denominador: total de partes iguais.
  • Frações equivalentes representam o mesmo valor.
  • Frações podem ser próprias, impróprias e aparentes.

Operações com frações

a/b + c/b = (a+c)/b a/b × c/d = ac/bd a/b ÷ c/d = a/b × d/c

4. Potências e raízes

Potenciação

aⁿ = a × a × a × ... (n vezes)
  • Potência negativa: a⁻ⁿ = 1 / aⁿ
  • Potência zero: a⁰ = 1, com a ≠ 0
  • Produto de potências: aᵐ · aⁿ = aᵐ⁺ⁿ

Radiciação

√a = b, se b² = a ⁿ√a = b, se bⁿ = a
  • Raiz quadrada: índice 2.
  • Raiz cúbica: índice 3.
  • Relacionar raiz e potência inversa.

5. Álgebra

Expressões algébricas

Uso de letras para representar valores desconhecidos ou variáveis.

2x + 3

Equação do 1º grau

ax + b = 0

Exemplo: 2x + 4 = 0 → x = -2

Equação do 2º grau

ax² + bx + c = 0 x = (-b ± √(b² - 4ac)) / 2a

6. Geometria

Formas planas

  • Triângulo
  • Quadrado
  • Retângulo
  • Círculo

Perímetro e área

Quadrado: A = l² Retângulo: A = b × h Triângulo: A = (b × h) / 2 Círculo: A = πr²

Volume

Cubo: V = a³ Paralelepípedo: V = c × l × h

7. Grandezas e medidas

Unidades importantes

  • Comprimento: mm, cm, m, km
  • Massa: mg, g, kg
  • Capacidade: mL, L
  • Tempo: s, min, h

Conversões

1 m = 100 cm 1 km = 1000 m 1 kg = 1000 g 1 L = 1000 mL

8. Estatística e probabilidade

Estatística

  • Média aritmética
  • Moda
  • Mediana
  • Leitura de tabelas e gráficos
média = soma dos valores / quantidade

Probabilidade

P(A) = casos favoráveis / casos possíveis

Exemplo: chance de sair número par em um dado = 3/6 = 1/2
